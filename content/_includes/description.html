Moving the Selector slider changes four values:

  • The index number for the UTF-8 character to display. (There are 297,334 characters. Technically they're different from Unicode, but we're gonna ignore that.)
  • An oklch() lightness value between 50-90%.
  • An oklch() chroma value between 0.08-0.18.
  • An oklch() hue value between 0-360.

Collectively it assigns a unique color to each character that change as you move the selector.
